### Tests klasei Kalkulators

**Klase Calculator**

Pa soļiem:

1. Izveidojiet klasi Calculator
```java
public class Calculator {

    public int sum(int a, int b) {
        return a + b;
    }

    public int subtract(int a, int b) {
        return a - b;
    }

    public int multiply(int a, int b) {
        return a * b;
    }

    public int divide(int a, int b) {
        return a / b;
    }

}
```
